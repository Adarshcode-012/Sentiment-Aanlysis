# AI-Enabled Market Trend & Consumer Sentiment Forecaster – Frontend

## Overview
This repository contains the **frontend dashboard** for the AI-Enabled Market Trend & Consumer Sentiment Forecaster project.

The frontend provides an interactive interface where users can explore **consumer sentiment, market trends, and AI-generated insights** derived from analyzed data.

The goal of this interface is to present **complex consumer analytics in a clear and visually intuitive way** so that marketing teams, analysts, and decision makers can easily understand trends and sentiment patterns.

---

## Key Features

### 1. Dashboard
The dashboard provides a high-level overview of consumer insights including:

- Sentiment distribution (positive, negative, neutral)
- Trend indicators for consumer discussions
- Summary statistics of analyzed data

---

### 2. Sentiment Analysis
This section visualizes the **sentiment classification results** obtained from the backend machine learning pipeline.

It displays:

- Sentiment distribution charts
- Comparative sentiment trends
- Consumer opinion patterns

---

### 3. Topic Insights
This section highlights **emerging discussion topics** identified using topic modeling techniques.

Users can explore:

- Trending topics
- Topic clusters
- Market discussion themes

---

### 4. Insights Assistant
The Insights Assistant provides **AI-driven contextual insights** derived from the analyzed data.

Users can:

- Ask questions about consumer trends
- Retrieve summarized insights from the dataset
- Explore deeper context through AI-based responses

---

## Current Development Status
At the current stage, the dashboard uses **mock data** to demonstrate the UI design and visualization structure.

The backend pipeline already generates:

- Cleaned datasets
- Sentiment analysis results
- Topic modeling outputs

These outputs will be integrated with the frontend in the upcoming milestone to enable **real-time dynamic analytics dashboards**.

---
